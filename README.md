# Java 多功能桌面辅助插件 (Desktop Plugin)

基于 Java Swing 和 FlatLaf 开发的桌面多功能时间与天气辅助工具。本项目非常适合作为课程设计或小型毕业设计，提供直观的深色 UI 与实用的日常功能。

## 🌟 核心功能
* **时钟与日历**：主界面实时突出显示当前时间、日期、星期以及农历，帮助用户轻松把握时间节奏。
* **倒数日提醒**：支持自定义目标事件（例如：期末考试）及目标日期，主界面清晰展示目标剩余天数。
* **天气预报展示**：集成【心知天气】API，自动获取配置城市的当天及未来两天的详细天气信息（包含最高/最低温度与直观的天气状态图标）。
* **内置闹钟与定时关机**：
  * 支持自定义闹钟时间，触发后弹出窗口居中置顶提醒。
  * 支持利用 Windows 系统命令实现定时自动关机动作配置，带有前置倒计时提醒界面。

## 📸 功能演示
- **主界面 (时间/天气/倒数日)**：涵盖了主要信息展示。
- **设置界面**：用于快速设置倒数目标、城市以及配置项。
- **提醒界面**：包含闹钟弹窗和关机倒计时弹窗交互提醒。

## 🛠 技术栈
* **开发语言**：Java (JDK 8 或以上版本)
* **GUI 框架**：Java Swing 主流窗体开发，并辅以 `JFormDesigner` 进行可视化界面设计。
* **UI 主题**：使用开源 [FlatLaf](https://github.com/JFormDesigner/FlatLaf) (扁平化深色主题 `FlatDarkLaf`)。
* **JSON 数据解析**：引入 [FastJSON](https://github.com/alibaba/fastjson) 用于解析天气 API 返回的层级响应。
* **第三方服务**：[心知天气 v3 API](https://www.seniverse.com/)。

## 🚀 部署与运行方法

### 1. 环境准备
* 确保系统已安装 Java 运行环境（JRE/JDK）。
* 推荐使用 IntelliJ IDEA 导入该项目，并确保 `lib/` 目录下的依赖项已被引入到项目中(Add as Library)。

### 2. 获取天气 API Key
本项目天气数据来源于**心知天气**平台，你必须配置自己的访问私钥：
1. 前往 [心知天气官网](https://www.seniverse.com/) 免费注册账号。
2. 在控制台创建一个免费版 API 访问凭据，获取该密钥字符串（Key）。

### 3. 修改配置文件
应用的本地配置均存储于项目根目录的 `src/config.txt` 中。在使用前，请打开该文件并按固定对应格式调整：
```text
待办事项;目标日期(yyyy/M/d);城市;你的API私钥;是否打开闹钟(true/false);闹钟时间(HH:mm);是否打开关机(true/false);关机时间(HH:mm);
```
**示例：**
```text
期末考试;2024/8/16;北京;YOUR_SENIVERSE_API_KEY;false;17:48;true;17:50;
```

### 4. 运行程序
找到 `src/GUI_Test.java` 类，运行其中的 `main` 方法即可启动程序主体。在主界面中点击文本对应区域，即可呼出配置面板等模块窗口。

## 📁 目录结构说明
- `src/`：程序源代码及其静态配套资源
  - `GUI_Test.java`：项目主入口与主体界面逻辑控制台。
  - `HttpURLConnectionUtil.java`：原生实现的 HTTP 网络请求处理工具类。
  - `LunarUtil.java`：农历与公历日期的相关转换计算工具。
  - `config.txt`：软件状态配置持久化存储库。
  - `image/`：程序所需要的天气图标等各类本地静态资源。
- `lib/`：项目相关 jar 依赖库存放路径。

## 📌 额外体验注意事项
* **网络请求频率**：天气的网络数据更新被限定在每 5 分钟的整数倍时段发送检测请求（为防 API 请求过度消耗），需确保测试环境网络联通良好。
* **自动关机隐患**：关机功能由底层执行 `shutdown -s -t 20` 触发，请在开发调试中慎重启用，可以通过拦截对应方法并只显示弹窗以作单纯的流程演示。
